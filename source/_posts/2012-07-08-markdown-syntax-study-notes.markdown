Markdown的语法还是相当简单的，这篇博客分享一下基本的语法，以后在工作中的一些随手笔记也可以用markdown语法来写，然后快速成文博客了。

##标题
标题用`#`开头，一个`#`是一级标题，同理`##`是二级标题，以此类推。`#`越多字体越小

##链接
	[text](link)
图片链接类似，区别是前面多一个叹号`!`
	![text](link)
	
##引用
用`>`开始一个段落，这个段落就会显示为引用

>我引用你啦！！！我引用你啦！！

##文字效果
斜体：用`*`或者`_`包围

*斜体斜体*

粗体：用`**`或者`__`包围

**粗体**

##列表
用`*`,`+`,`-`开头，子项缩进两个空格

##代码块
很简单，开头空四个空格，或者一个tab键，所在内容就会以代码块形式显示。

	#include<iostream>
	int main() {
		return 0;
	}
同样用`包围的文字也会以代码格式显示。

##分割线
在一行中用三个或以上的`*`号或者`-`号，就能建立起一个分割线。注意：行内不能有空白以外的其它东西。
	* * *
	***
	******
	- - - 
	-----------------------

***
